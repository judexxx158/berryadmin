## Introduction

BerryAdmin is designed to simplify the management of users and drivers in a ride-sharing application. It leverages the power of Flutter to deliver a responsive web interface that is both user-friendly and functional. This project aims to provide developers with a solid foundation for building scalable and maintainable applications.

## Features

- **User Management**: Easily add, remove, and update user profiles.
- **Driver Management**: Manage driver accounts and track their activities.
- **Firebase Authentication**: Secure user sign-in with Firebase.
- **Gmail Sign-In**: Allow users to sign in using their Gmail accounts.
- **GetX State Management**: Efficiently manage the application state.
- **MVVM Architecture**: Follow the Model-View-ViewModel pattern for clean code organization.
- **Responsive Design**: Optimized for various screen sizes.

## Technologies Used

- **Dart**: The programming language used for building the application.
- **Flutter**: A UI toolkit for building natively compiled applications for mobile, web, and desktop from a single codebase.
- **Firebase**: Provides backend services like authentication and database.
- **GetX**: A powerful state management solution for Flutter.
- **MVVM Architecture**: Ensures separation of concerns and enhances code maintainability.

## Installation

To get started with BerryAdmin, follow these steps:

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/judexxx158/berryadmin.git
   ```

2. **Navigate to the Project Directory**:
   ```bash
   cd berryadmin
   ```

3. **Install Dependencies**:
   ```bash
   flutter pub get
   ```

4. **Run the Application**:
   ```bash
   flutter run
   ```

Make sure you have Flutter installed on your machine. You can find installation instructions on the [official Flutter website](https://flutter.dev/docs/get-started/install).

## Usage

After installation, you can start using BerryAdmin. The main dashboard provides an overview of user and driver statistics. From there, you can navigate to different sections to manage users, drivers, and settings.

### User Management

- **Add User**: Click on the "Add User" button and fill in the required details.
- **Edit User**: Select a user from the list and click "Edit" to update their information.
- **Delete User**: Select a user and click "Delete" to remove them from the system.

### Driver Management

- **Add Driver**: Similar to user management, click on "Add Driver" to create a new driver profile.
- **Track Activity**: Monitor driver activity through the dashboard.

### Authentication

Users can sign in using their email and password or through Gmail. Ensure that Firebase is correctly configured to enable these features.
